🏛️ Patroclus Concierge Integration Guide

Step 1: Add Scripts to Your Main HTML

<!-- Add before closing </body> tag -->
<script src="concierge-config.js"></script>
<script src="gemini-concierge.js"></script>

Step 2: Configure API Key

For Production:

<!-- Add to <head> section -->
<meta name="gemini-api-key" content="YOUR_ACTUAL_GEMINI_API_KEY">

For Development:

The system will prompt for the API key on localhost and store it securely.

Step 3: Test the Integration

  1. Load your website with the new scripts
  2. Press 'C' to open the chat (existing B-12 functionality)
  3. Type a message to test the Gemini concierge response
  4. Check browser console for initialization status

Features

  • ✅ Professional Gemini-powered responses
  • ✅ Graceful fallback to intelligent responses if API unavailable
  • ✅ Context-aware navigation assistance
  • ✅ Maintains existing B-12 UI/UX
  • ✅ Professional typing indicators
  • ✅ Security-first API key management
  • ✅ Error handling with dignity preservation

Debugging

Open browser console and type: getConciergeStatus()
